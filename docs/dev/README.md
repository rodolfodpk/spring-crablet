Material here is for **maintainers and contributors** working on the Crablet **codebase** itself.

**End users and application teams** should start from the root [README](../../README.md) and the
guides under `docs/user/` (quickstart, tutorials, configuration, etc.) — not from this folder.

| Path | Contents |
|------|----------|
| [`DOCS_VERIFICATION.md`](DOCS_VERIFICATION.md) | What `make docs-check` validates |
| [`PRODUCT_ROADMAP.md`](PRODUCT_ROADMAP.md) | AI-assisted codegen and Kubernetes generation status — the only place those pré-1.0 tracks are documented |

Historical design notes, completed migration write-ups, and past reviews are not kept as
standalone docs — the Git history is the archive for that material.
